# prisma DB Push

将本地的 prisma/schema.prisma 变更，‘应用’到数据库中。

npx prisma db push

# prisma DB Pull

将直接修改[create | alter table]数据库的结果，同步到prisma/schema.prisma

npx prisma db pull
